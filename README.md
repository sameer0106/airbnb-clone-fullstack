# 🏡 Airbnb Clone

A web-based Airbnb clone built using Node.js, Express.js, MongoDB, and EJS following the MVC architecture. The application allows users to explore property listings, manage accounts, and interact with features like favorites and bookings.

---

## 🚀 Features

- 🔐 Role-based Authentication (Guest & Host)
- 🏠 Create, edit, and delete property listings
- ❤️ Add/remove properties to favorites
- 🧾 Session-based login system
- 📂 MVC structured backend
- 📊 MongoDB database integration (Mongoose)

---

## 🛠️ Tech Stack

- **Backend:** Node.js, Express.js  
- **Frontend:** EJS, CSS  
- **Database:** MongoDB (Mongoose)  
- **Authentication:** Sessions & Cookies

## ⚙️ Installation & Setup

1. Navigate to project folder

```bash
cd airbnb-clone-mern
```

2. Install dependencies

```bash
npm install
```

3. Create a `.env` file and add your MongoDB URI

4. Run the server

```bash
npm start
```
